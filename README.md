# [MASTER 24](https://24.metamemo.uk/)

24点大师是一个在线工具，帮助你解决24点游戏的问题。只需输入你的四个数字，就能得到解决方案。

## 使用方法

1. 打开网页。
2. 在提供的输入框中，输入你的四个数字。
3. 点击 "Calculate" 按钮。
4. 查看可能的解决方案。
